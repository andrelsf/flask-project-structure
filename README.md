# Flask RESTful

Learning about Flask RESTful and

### VirtualEnv

```bash
pip3 install virtualenv
virtualenv -p python3.6 env
```

### Install

```bash
pip3 install flask-restful
```

### Running APP

```bash
export FLASK_ENV=development
export FLASK_APP=app.py
python3.6 app.py
```
